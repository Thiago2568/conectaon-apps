Como instalar no Windows

  1. Extraia o ZIP antes de abrir. Clique com o botão direito no arquivo baixado e escolha "Extrair Tudo" para uma pasta fixa, por exemplo C:\SintegraView\.
  2. Abra a pasta extraída e dê dois cliques em SintegraView.exe.
  3. O navegador abre sozinho com a interface. Deixe a janela preta aberta enquanto usa.
Importante: nunca execute o programa direto de dentro do ZIP. O Windows abre o arquivo numa pasta temporária e seus dados podem ser perdidos. Sempre extraia primeiro.

Como instalar no Mac

  1. Extraia o ZIP (dois cliques) e mova o SintegraView para uma pasta fixa, por exemplo a pasta Aplicativos ou ~/SintegraView.
  2. Na primeira vez, clique com o botão direito no arquivo e escolha Abrir (necessário porque o app não é assinado pela Apple). Confirme em "Abrir" na janela que aparecer.
  3. O Terminal abre e o navegador carrega a interface sozinho. Deixe o Terminal aberto enquanto usa.

Como usar

  1. Importar: arraste os arquivos direto para a tela (DACON em PDF, Sintegra em TXT, EFD-Contribuições em TXT/SPED).
  2. Auditoria: um clique roda as validações automáticas.
  3. Dashboard: resumo visual de bonificações, créditos e diferença.
  4. Exportar: baixe os relatórios em CSV.

Seus dados ficam apenas no seu computador, na pasta SintegraView/bancos. Nada é enviado para a internet.
